Библиотека для работы с файлами формата Teledisk

Автор: Panther (http://www.emuverse.ru/wiki/User:Panther)

Библиотека, данная программа и их исходные коды распространяются под
лицензией GNU General Public License, текст которой находится
в файле LICENSE.TXT

Файлы:
teledisk10.pas		- Исходный код
tdlzss10.pas		- Исходный код для распаковки данных LZSS
td2dsk.dpr		- Тестовый проект для демонстрации использования
			  библиотеки. Конвертер .TD0 в посекторный образ.
dsk2td.dpr		- Обратный конвертер посекторных образов в .TD0
exe/			- Откомпилированные примеры, текст лицензии.
